<summary><b>思路1</b></summary>

- 先计算链表长度 `L`，再重新走 `L-k` 步；

<details><summary><b>Python</b></summary>

```python
class Solution:
    def FindKthToTail(self , pHead: ListNode, k: int) -> ListNode:
        
        L = 0
        cur = pHead
        while cur:
            cur = cur.next
            L += 1
        
        cur = pHead
        d = L - k
        while d and cur:
            cur = cur.next
            d -= 1
        
        return cur
```

</details>


<summary><b>思路2：快慢指针</b></summary>

- 快指针先走 `k` 步；最后返回慢指针；

<details><summary><b>Python</b></summary>

```python
class Solution:
    def FindKthToTail(self , pHead: ListNode, k: int) -> ListNode:
        
        f, s = pHead, pHead
        # 快指针先走 k 步
        for _ in range(k):
            if not f:
                return f
            f = f.next
        
        while f:
            f = f.next
            s = s.next
        
        return s
```
